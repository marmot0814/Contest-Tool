# Contest Tool

## Requirement
- docker-compose

## Build Environment
```
git clone https://github.com/marmot0814/Contest-Tool.git
cd Contest-Tool

make build
```

## Add a problem
```
python script/folderGenerator.py {PROBLEM_NAME}
```

## Generate pdf
```
make pdf
```

## Compress DOMjudge format zip file
```
make compress
```
